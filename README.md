# HandTracking Program for Art Softwares
Some drawing tablets arenâ€™t equipped with the best shortcut buttons in addition to some not even having any of these buttons including scrolling options, or mouse artists like myself back in the day spent too much time dragging the mouse all over the screen to flip the screen. This program aims to allow users and artists to use hand motions to use shortcuts based on art programs (pc only), as well as allowing the user to modify whichever motion to their desired shortcuts. This also works on any program including games.
7 Hand gestures - Pointer, Peace sign, Fist, Thumbs up, Pinch in, Pinch out, Rocker sign, 
Can have keybinds repeat and scroll ( scroll value is customizable by pixel amount )
